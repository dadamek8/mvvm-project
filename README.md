# MVVM Desktop Application (.NET, WPF)

Desktop application built with C# and WPF using MVVM architecture.
The project demonstrates separation of concerns, data binding, and interaction with SQL database.

## Features

- MVVM architecture
- CRUD operations
- SQL Server integration
- Data validation
- Dependency separation
- Clean code structure

## Architecture

Project uses MVVM pattern:

Model – data layer  
View – UI layer  
ViewModel – business logic and state management  

Project structure:

/Models  
/ViewModels  
/Views  
/Services  

## How to run

Clone repo
Open in Visual Studio
Update connection string
Run database migration
Start application
